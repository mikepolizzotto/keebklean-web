Frequently asked

How do I unlock?

Press and hold anywhere on the lock screen until the ring fills (3–15 seconds, set in Settings). Let go early and it stays locked. You can also hold your chosen unlock key — the Escape key by default.

Some keys still work on the App Store version

Apple's App Store sandbox doesn't allow any app to intercept certain hardware keys (function keys, media keys, Caps Lock). Everything else — typing, shortcuts, app-switching — is locked. If you want to lock every key, use the Direct version, which uses an Accessibility permission to do so.

The Direct version asks for Accessibility permission

That's expected. KeebKlean needs Accessibility access to block keyboard input while a lock is active. Grant it under System Settings → Privacy & Security → Accessibility, then relaunch the app. It's never used to read or record what you type.

Is my data safe?

Completely. KeebKlean collects nothing and never connects to the internet. See our Privacy Policy.
