# keebklean.com

Marketing + privacy/support site for **KeebKlean** (the Mac app). Static HTML, served via
GitHub Pages at https://keebklean.com.

- `index.html` — landing page
- `privacy.html` — privacy policy (App Store required URL)
- `support.html` — support + FAQ
- `style.css` — styles
- `CNAME` — custom domain for GitHub Pages

The app source lives in a separate private repo.
